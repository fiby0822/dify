# Dify変数参照クイックリファレンス

## 場所別の正しい記法

### 1. template-transformのtemplate内
```yaml
template: |
  ユーザー入力：{{ sys.query }}
  データ：{{ data_variable }}
  フィルタ付き：{{ result | trim | tojson }}
```

### 2. tool_parametersのvalue
```yaml
tool_parameters:
  data:
    type: mixed
    value: '{{#transform_node.output#}}'
```

### 3. LLMのprompt_template
```yaml
prompt_template:
  - role: user
    text: "入力：{{#sys.query#}}"
```

### 4. answerノードのanswer
```yaml
answer: |
  結果：{{#llm_node.text#}}
```

## よくある間違いと修正方法

✅ 修正：modelを使用
