# Dify YMLファイル作成依頼プロンプト

## 要件
LLMコンポーネントの記述ルールが正しく機能することを検証するため、以下の要件を満たすDifyアプリケーションのYMLファイルを作成してください：

1. **LLMノードの基本動作確認**
   - systemロールとuserロールの両方を使用
   - 変数参照（{{#node_id.variable#}}形式）を含む
   - 適切な温度設定を使用

2. **必須パラメータの検証**
   - すべての必須パラメータが正しく設定されていること
   - structured_output_enabled: false
   - vision.enabled: false
   - variables: []
   - height/width/position関連の設定

3. **実際の使用ケース**
   - ユーザーからの質問を受け取り、分析・回答するシンプルなチャットボット
   - 開始ノード → LLMノード → エンドノードの基本的な流れ
