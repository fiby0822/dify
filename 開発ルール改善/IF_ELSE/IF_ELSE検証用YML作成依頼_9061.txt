# Dify YMLファイル作成依頼プロンプト

## 要件
ユーザーの年齢に基づいて異なるメッセージを返すワークフローを作成してください。

### 詳細要件：
1. **年齢入力ノード**：ユーザーから年齢を数値として入力を受け取る
2. **年齢判定ノード（IF_ELSE）**：年齢に基づいて以下の3つのケースに分岐
   - ケース1（未成年）：20歳未満の場合
   - ケース2（成人）：20歳以上60歳未満の場合
   - ケース3（シニア）：60歳以上の場合
3. **各ケースの処理**：
   - 未成年：「未成年の方向けのサービスをご案内します」と返答
   - 成人：「成人の方向けの標準サービスをご案内します」と返答
   - シニア：「シニア世代向けの特別サービスをご案内します」と返答
4. **最終出力**：各ケースで生成されたメッセージを最終的に表示

### 技術的要件：
- IF_ELSEノードを使用した条件分岐の実装
- 数値比較演算子（"<", ">="）を使用
- 各ケースには必ず条件IDを設定
- case_idとidの両方を必ず設定
- エッジのsourceHandleとcase_idを正しく対応させる
