# Dify YMLファイル作成依頼プロンプト

## 要件
顧客サポートチケット振り分けシステムを作成してください。

### 機能要件
1. 顧客からの問い合わせ内容を受け取る
2. 問い合わせ内容のキーワードを基に適切な部署に振り分ける
   - 「エラー」「動作しない」「不具合」「故障」などのキーワード → 技術サポート部門
   - 「料金」「請求」「支払」「価格」などのキーワード → 請求関連部門
   - その他 → 一般問い合わせ部門
3. 緊急度の判定を行う
   - 「緊急」「至急」「すぐに」などのキーワードがある場合 → 高優先度
   - それ以外 → 通常優先度
4. 振り分け結果と対応優先度を含むメッセージを出力する

### 制約事項
- Codeノードは使用しないこと
- IF_ELSEノードを使用して条件分岐を実装すること
- 各条件は明確に定義し、重複しないようにすること
