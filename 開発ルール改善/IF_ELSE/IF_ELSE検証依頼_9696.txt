# Dify YMLファイル作成依頼プロンプト

## 要件
顧客問い合わせ自動分類システムを作成してください。

### 処理フロー
1. **顧客からの問い合わせ受付**
   - ユーザーから問い合わせ内容を受け付ける

2. **問い合わせ内容の分類（IF_ELSEノード使用）**
   - 技術的な問題：「エラー」「不具合」「動作しない」を含む場合
   - 料金・請求関連：「料金」「請求」「支払い」を含む場合  
   - 製品情報：「仕様」「機能」「性能」を含む場合
   - その他：上記に該当しない場合（デフォルトケース）

3. **各分類に応じた独立した回答生成**
   - 各分岐で独立したLLMノードを使用して専門的な回答を生成
   - 独立実行パターンを採用（各分岐が独立して完結）

### 技術要件
- **必須**：version: 0.3.0を使用
- モード：advanced-chat
- ノードタイプ：基本4種類（start、llm、if-else、answer）のみ使用
- Codeノードは使用しない
- 比較演算子はASCII文字のみ（>=、<=、!=）を使用
- 文字列検索（contains）を優先的に使用
- 各ケースにidとcase_idを設定（同じ値）
- sourceHandleはcase_idまたは"false"を使用
- targetTypeは実際のノードタイプを指定（customは使用しない）
