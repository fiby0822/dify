# Dify YMLファイル作成依頼プロンプト

## 要件
IF_ELSEコンポーネントの動作検証用YMLファイルを作成してください。以下の機能を実装してください：

1. **ユーザー入力の受付**
   - スコア（数値）の入力
   - ステータス（文字列）の入力

2. **IF_ELSE条件分岐の実装**
   - スコアによる3段階の分岐（低得点：30以下、中得点：31-70、高得点：71以上）
   - 各分岐で異なるメッセージを生成
   - Unicode比較演算子（≤、≥）を使用した数値比較
   - case_idフィールドの必須設定の確認

3. **分岐後の処理**
   - 各分岐で独立したメッセージを生成
   - 最終的に適切なメッセージを出力
   - 推奨パターン1（独立answer型）を採用

4. **エラーケースの検証**
   - 無効な入力（数値以外、範囲外等）への対応
   - デフォルトケースの動作確認
