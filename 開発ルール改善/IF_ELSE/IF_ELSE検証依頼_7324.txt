# Dify YMLファイル作成依頼プロンプト

## 要件
IF/ELSEコンポーネントの動作検証用YMLファイルを作成してください。
以下の機能をすべて検証できるワークフローを実装してください：

### 検証項目
1. **単一条件の検証**
   - is（完全一致）演算子のテスト
   - is not（不一致）演算子のテスト
   - contains（部分一致）演算子のテスト
   - not contains（部分不一致）演算子のテスト

2. **数値比較の検証**
   - 大なり（>）演算子のテスト
   - 小なり（<）演算子のテスト
   - 以上（>=）演算子のテスト
   - 以下（<=）演算子のテスト

3. **複数条件の検証**
   - AND論理演算子を使った複合条件
   - OR論理演算子を使った複合条件

4. **エッジケースの検証**
   - いずれの条件にも該当しない場合の処理
   - デフォルトケースの動作確認

### 実装シナリオ
「ユーザー情報の検証システム」として実装してください：
- 入力：ユーザーの年齢、会員種別、地域、優先度スコア
- 処理：各種条件に基づいて、適切なメッセージや処理ルートに分岐
- 出力：各条件に応じた検証結果メッセージ
