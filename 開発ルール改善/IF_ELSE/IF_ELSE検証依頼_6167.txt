# Dify YMLファイル作成依頼プロンプト

## 要件
従業員の勤怠状況判定システムを作成してください。

### 機能要件
1. 従業員の出勤時間を入力（HH:MM形式、例：08:30）
2. 出勤時間に基づいて勤怠状況を判定
   - 早朝出勤：7:59以前（07:00-07:59など）
   - 通常出勤：8:00-8:59（08:00-08:59）
   - 遅刻：9:00以降（09:00以降）
3. 遅刻の場合、さらに程度を判定
   - 軽微な遅刻：9:00-9:29（09:00-09:29）
   - 重大な遅刻：9:30以降（09:30以降）
4. 判定結果と対応指針を含むメッセージを出力

### 実装上の注意事項
- Codeノードは使用しないこと
- IF_ELSEノードを使用して条件分岐を実装すること
- 時刻の比較は文字列比較で行うこと（HH:MM形式の文字列は辞書順で比較可能）
- 各条件は明確に定義し、重複しないようにすること
- version: 0.3.0を必ず使用すること
