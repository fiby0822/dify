# Dify YMLファイル作成依頼プロンプト

## 要件
最新情報検索アシスタントの開発をお願いします。

### 機能要件
1. ユーザーから検索キーワードや質問を受け付ける
2. WEB検索（Tavily Search）を使って最新情報を検索する
3. 検索結果をAIが要約・分析する
4. ユーザーに分かりやすい形で回答を提供する

### 技術要件
- Codeノードは使用しない
- WEB検索にはTavily Searchツールを使用
- 検索結果の要約にはLLMノードを使用
- advanced-chatモードで実装

### 検索オプション
- 検索深度：basic（標準）
- 最大結果数：5件
- 検索カテゴリ：general（一般）
- 回答形式：要約された情報として提供

### ユーザビリティ要件
- 開始時のメッセージで使い方を説明
- サジェスト質問を3つ用意
- エラー時は適切なメッセージを表示
