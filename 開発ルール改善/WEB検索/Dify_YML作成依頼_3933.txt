# Dify YMLファイル作成依頼プロンプト

## 要件
最新のAI技術に関するニュースを検索して要約するワークフローを作成してください。

### 機能要件
1. ユーザーからAI技術に関する検索キーワードを受け取る
2. Tavily SearchツールでWEB検索を実行
3. 検索結果を要約して分かりやすく表示する
4. version: 0.3.0 を使用すること
5. Codeノードは使用しない

### ワークフロー構成
- Start → WEB検索（Tavily Search） → LLM（要約処理） → Answer
